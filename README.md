# GraphQL Profile Dashboard

⚠️ Archived project (early GraphQL & frontend learning)

Web application that authenticates users and displays personalized profile data using **GraphQL** and **JWT authentication**, with statistics visualized using **SVG**.

## Highlights
- GraphQL queries (simple, nested, arguments)
- JWT-based login & logout
- User profile dashboard
- SVG-based statistic graphs
- Vanilla JavaScript, HTML, CSS

## Tech Stack
JavaScript · GraphQL · HTML · CSS · SVG · JWT
